# PRD：儿童国际音标识读辅助工具（H5）

## 1. 文档信息

- 文档名称：儿童国际音标识读辅助工具 PRD
- 文档版本：v1.0
- 创建日期：2026-07-10
- 产品形态：移动端 Web / H5
- 目标阶段：MVP / 首版

## 2. 项目背景

家长或老师在陪孩子学习英语时，经常会遇到一个高频问题：孩子能看到单词，但很难稳定建立“字母或字母组合”和“国际音标发音”之间的对应关系。

尤其是在真实学习材料中，例如绘本、单词卡、练习纸、屏幕截图，单词并不是按音标规律组织好的，家长往往需要手工摘词、查音标、再自己总结规律，过程耗时且不稳定。

本产品希望解决的问题是：

1. 让家长/老师通过拍照或上传图片，快速导入图片中的英文单词。
2. 自动为导入单词生成统一的英式 IPA 音标，并建立可查询的音标库。
3. 支持从单个音标反向查找所有相关单词，并展示单词中与该音标对应的字母或字母组合。
4. 以醒目的方式拆解单词中的字母组合和音标配对，帮助孩子形成稳定的音形关联认知。

## 3. 产品目标

### 3.1 核心目标

打造一个“拍照即建库”的教学辅助工具，帮助家长/老师从真实材料中快速提取可讲解的音标学习样本。

### 3.2 首版目标

首版只聚焦以下闭环：

`拍照/上传图片 -> OCR 提取单词 -> 人工轻校对 -> 自动生成英式 IPA 音标库 -> 按音标反查单词 -> 展示字母组合与音标对应关系`

### 3.3 非目标

首版不追求成为完整英语学习平台，不覆盖以下方向：

- 完整课程体系
- 多口音切换
- 发音打分
- 手写体识别
- 多人协作教学
- 重度游戏化系统

## 4. 目标用户

### 4.1 主要用户

家长 / 老师。

首版默认由大人主导操作，孩子主要负责观看、辨认、练习。

### 4.2 次级使用者

孩子。

孩子在首版中的主要角色不是独立操作者，而是被引导的学习者。因此首版更强调：

- 结果清晰
- 高亮明显
- 拆解直观
- 可边讲边看

## 5. 使用场景

### 5.1 典型场景

1. 家长拍摄一本绘本或单词卡页面，快速提取其中的英文单词。
2. 老师导入一张课堂 PPT 截图，整理出其中的目标词汇。
3. 家长围绕一个音标，例如 `/f/`，快速找到当前词库里的所有相关单词。
4. 家长向孩子讲解某个单词时，直接展示该单词里的字母组合和音标对应关系，例如 `ph -> /f/`。

### 5.2 支持的图片来源

- 手机直接拍照
- 相册图片
- 屏幕截图

### 5.3 首版暂不支持

- 手写英文识别
- 复杂版式下的段落级解析
- 非英文为主的混排内容深度理解

## 6. 产品定位

这是一个“词库优先型”的音标识读辅助工具，而不是“练习优先型”或“游戏优先型”产品。

首版优先级排序：

1. 导入真实材料中的英文单词
2. 建立可查询的音标库
3. 建立单词内部的音标与字母组合关联
4. 为家长/老师提供高可解释性的展示界面
5. 在此基础上补充轻量练习与趣味化机制

## 7. 产品原则

1. 真实材料优先：练习素材来自用户刚导入的图片内容。
2. 词库结构化优先：不是只识别文本，而是生成可查询、可教学的结构化词库。
3. 可解释性优先：展示“为什么这个单词会出现在这个音标查询结果里”。
4. 轻校对优先：允许 OCR 不完美，但必须让用户快速修正。
5. 教学友好优先：界面不是偏后台管理，而是适合边讲边看。

## 8. 用户流程

### 8.1 主流程

1. 用户进入首页。
2. 用户点击“拍照导入”或“上传图片”。
3. 系统执行 OCR，提取候选英文单词。
4. 系统对识别结果做清洗、去重、过滤。
5. 用户快速删除误识别词，并可手动补充漏词。
6. 系统为确认导入的单词生成英式 IPA 音标和结构化索引。
7. 用户进入音标库页面，按音标或单词查看结果。
8. 用户点击单个音标，查看所有相关单词及其对应字母组合。
9. 用户点击单词，查看字母组合和音标的成对拆解展示。

### 8.2 成功标准流程

用户可以在 3 分钟内完成以下任务：

- 导入一张图片
- 获得一组较干净的英文单词
- 成功生成音标库
- 围绕一个音标快速找到例词并开展讲解

## 9. 信息架构

首版建议采用 4 个核心页面 / 视图：

1. 导入页
2. 识别校对页
3. 词库 / 音标库页
4. 单词详情 / 拆解展示页

### 9.1 导入页

功能：

- 拍照导入
- 上传图片
- 查看当前导入状态

### 9.2 识别校对页

功能：

- 展示 OCR 提取出的候选单词
- 删除误识别项
- 手动补录单词
- 去重后确认导入

### 9.3 词库 / 音标库页

功能：

- 查看已导入单词列表
- 查看音标列表
- 点击某个音标查看相关单词
- 按音标搜索

### 9.4 单词详情 / 拆解展示页

功能：

- 查看完整单词
- 查看完整英式 IPA
- 查看音节切分
- 查看字母组合和音标的成对展示
- 高亮当前讲解片段

## 10. 核心功能需求

### 10.1 图片导词

#### 功能描述

支持用户通过拍照或上传图片的方式导入包含英文单词的内容。

#### 输入

- 拍摄图片
- 相册图片
- 屏幕截图

#### 输出

候选英文单词列表，而不是整段文本原文。

#### 验收要点

- 支持移动端相机调用
- 支持上传本地图片
- 上传后能进入处理状态

### 10.2 OCR 识别与清洗

#### 功能描述

将图片中的文字识别结果清洗为可用单词列表。

#### 处理要求

- 提取英文单词
- 过滤明显的中文、数字、标点噪音
- 自动去重
- 合并大小写差异
- 标记低置信度词条

#### 验收要点

- 用户可以快速删除错误词条
- 用户可以手动补充漏识别单词
- 最终确认导入的是清洗后的单词集

### 10.3 音标生成

#### 功能描述

为确认导入的英文单词生成统一的英式 IPA 音标。

#### 要求

- 音标标准统一为英式 IPA
- 每个单词应有完整音标
- 生成可用于后续检索的音标片段序列
- 支持标记“音标生成失败”或“待确认”状态

#### 验收要点

- 大部分常见基础词可自动获得音标
- 对未命中的词有明确状态说明

### 10.4 音标库生成

#### 功能描述

根据导入单词自动建立音标索引库，支持按音标反查单词。

#### 核心能力

- 单词 -> 完整音标
- 单词 -> 音标片段
- 单个音标 -> 相关单词集合

#### 验收要点

- 点击某个音标后，能返回包含该音标的所有已导入单词

### 10.5 字母组合与音标配对

#### 功能描述

将单词内部的字母或字母组合，与音标片段进行可视化对应。

#### 示例目标

用户在 `phone` 中能明显看到 `ph -> /f/`。

#### 展示要求

- 成对显示
- 对齐展示
- 可高亮单个配对片段
- 对于不完全确定的配对，允许显示为“最相关字母组合”

#### 说明

英语中的字母组合与发音并非总是一一对应，首版以“教学上有帮助的高可解释性映射”为目标，不承诺语言学意义上的绝对精确。

### 10.6 音标查询

#### 功能描述

支持基于单个音标查看相关单词，并展示单词中的对应字母。

#### 查询形式

- 点击常见音标卡片
- 搜索一个音标

#### 查询结果展示

每个结果至少包含：

- 单词原文
- 完整英式 IPA
- 当前命中的音标
- 对应字母 / 字母组合高亮

### 10.7 单词拆解展示

#### 功能描述

将单词拆成可讲解的“字母组合 - 音标”配对单元，用于教学展示。

#### 展示原则

- 视觉上醒目
- 拆解结果一一对齐
- 当前高亮片段突出
- 适合家长边讲边指认

## 11. 数据模型建议

本节为产品层面数据结构建议，用于保证后续系统能支持查询与展示能力。

### 11.1 导入记录 Import

建议字段：

- `id`
- `source_type`：拍照 / 相册 / 截图
- `image_url`
- `ocr_raw_text`
- `candidate_words`
- `confirmed_words`
- `created_at`

### 11.2 单词记录 Word

建议字段：

- `id`
- `word`
- `normalized_word`
- `ipa_full`
- `syllables`
- `source_import_id`
- `status`：正常 / 待确认 / 已忽略

### 11.3 音标片段索引 WordPhoneme

建议字段：

- `id`
- `word_id`
- `phoneme`
- `position_in_word`
- `position_in_ipa`

用途：

- 支持“按单个音标查找相关单词”

### 11.4 字母-音标映射 GraphemePhonemeMap

建议字段：

- `id`
- `word_id`
- `grapheme`
- `phoneme`
- `start_index`
- `end_index`
- `confidence`

用途：

- 支持“显示对应字母组合”
- 支持后续统计“某个字母组合常见发音”

## 12. 查询逻辑

### 12.1 按单个音标查询单词

输入：

- 单个音标，例如 `/f/`

输出：

- 所有包含该音标的已导入单词
- 每个单词中最相关的字母或字母组合

### 12.2 按单词查看拆解

输入：

- 某个单词，例如 `phone`

输出：

- 完整英式 IPA
- 音节切分
- 字母组合与音标的配对展示

### 12.3 按字母组合查看常见发音

输入：

- 某个字母组合，例如 `ph`

输出：

- 在当前词库中，该字母组合最常关联的音标
- 相关例词

说明：

此能力建议首版作为弱功能预留，不一定作为主入口，但应在数据结构上支持。

## 13. 关键交互要求

### 13.1 导入反馈

用户上传图片后，系统需明确展示处理阶段，例如：

- 正在提取英文内容
- 正在清洗单词
- 正在生成音标库

### 13.2 校对体验

识别校对页应降低操作负担：

- 每个词条一键删除
- 顶部显示识别数量与建议导入数量
- 提供补录入口

### 13.3 查询体验

音标库页应允许用户低成本进入查询：

- 展示常见音标卡片
- 支持输入音标搜索
- 查询结果直接展示高亮关系

### 13.4 拆解展示体验

单词详情页应支持：

- 上下或左右对齐展示配对
- 单击某一段时同步高亮对应音标 / 字母组合

## 14. 错误处理与边界说明

### 14.1 OCR 不准确

可能问题：

- 混入非英文内容
- 单词切分错误
- 漏词

处理方式：

- 自动过滤无效内容
- 标记低置信度项
- 允许用户手动删除和补录

### 14.2 音标未命中

可能问题：

- 专有名词
- 异常拼写
- 词典未覆盖

处理方式：

- 标记为“待确认”
- 暂不写入正式音标库或以待校对状态写入

### 14.3 字母组合映射不绝对

说明：

英语中的音形对应不是严格一一映射。产品表达应避免绝对化：

- 不说“这个字母组合一定发这个音”
- 应说“在这个单词里，最相关的字母组合是 X”
- 或“在当前词库中，这个字母组合常见对应 Y”

### 15.1 包含

## 16. 测试设计建议

测试设计应围绕“当前已导入词库”展开，而不是脱离上下文随机出题。

### 16.1 音标找单词

系统给出一个音标，例如 `/f/`，让孩子从多个已导入单词中选出包含该音标的单词。

训练目标：

- 建立“音标 -> 例词”的反向联想

### 16.2 字母组合找发音

系统高亮某个字母组合，例如 `ph`，让孩子判断它最常对应哪个音标。

训练目标：

- 强化“字母组合 -> 常见发音”的规律认知

### 16.3 单词拆解配对

系统将字母片段和音标片段打乱，让孩子做匹配。

训练目标：

- 理解单词内部可以拆成若干音形配对单元

### 16.4 测试设计原则

- 只基于已导入词库出题
- 一次只练一个规律
- 即时反馈
- 错题可回流复习

## 17. 趣味化机制建议

首版建议采用轻量趣味化，而不是重度游戏化。

### 17.1 规律卡片

将常见字母组合发音规律做成卡片，例如：

- `ph -> /f/`
- `sh -> /ʃ/`
- `ee -> /i:/`

孩子掌握后可点亮对应卡片。

### 17.2 例词解锁

当某个音标下积累到一定数量的例词后，生成“我会这个音标了”卡片，展示：

- 对应音标
- 常见字母组合
- 代表单词

### 17.3 连续答对反馈

围绕同一规律设置连续答对反馈，例如：

- 连续答对 5 次 `ph -> /f/` 相关题目
- 提示孩子已经掌握该规律

### 17.4 亲子共学反馈

向家长展示过程型成果，例如：

- 今天新发现了多少组发音规律
- 当前图片整理出了多少个可教学单词
- 孩子已掌握多少个高频字母组合

### 17.5 首版建议纳入的趣味化能力

- 规律卡片
- 音标找单词
- 单词拆解配对

### 17.6 首版不建议纳入

- 排行榜
- 虚拟货币
- 复杂闯关地图
- 角色装扮系统
- 社交对战

## 18. 成功指标

首版建议关注以下指标：

### 18.1 任务完成指标

- 图片导入成功率
- OCR 后有效单词保留率
- 音标生成成功率
- 用户完成一次“导入 -> 查询”的闭环比例

### 18.2 教学使用指标

- 单次导入后平均被查看的单词数
- 音标查询使用次数
- 单词拆解页查看次数

### 18.3 学习互动指标

- 练习模块参与率
- 规律卡片点亮数量
- 错题回流后正确率提升

## 19. 后续迭代方向

在首版验证成立后，可按优先级继续扩展：

1. 听音选词
2. 看字母组合猜发音
3. 错题本
4. 复习节奏推荐
5. 家长查看“已掌握规律分布”
6. 美式 IPA 支持
7. 更复杂的句子级材料处理

## 20. 一句话定义

这是一个面向家长/老师的 H5 教学辅助工具，用拍照导词和自动音标建库的方式，把真实材料中的英文单词快速转化为可讲解的“字母组合 - 音标”关联样本，帮助孩子提升国际音标识读能力。
